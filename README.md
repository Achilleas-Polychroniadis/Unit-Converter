# Unit Converter
A simple JavaFX application for converting between units of length, weight, and temperature.

## Features
  - Supports conversions for:<br>
    - **Length:** Centimeters, Meters, Kilometers, Inches, Feet, Miles<br>
    - **Weight:** Grams, Kilograms, Pounds<br>
    - **Temperature:** Celsius, Fahrenheit, Kelvin<br>
  - User-friendly interface with real-time updates.<br>
